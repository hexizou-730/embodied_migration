# Archive

This folder keeps non-mainline materials so the repository root stays focused on
the current migration framework.

## Tracked Archive

| Folder | Contents |
|---|---|
| `legacy_docs/` | Older reports, setup notes, workshop framing, and document artifacts |
| `setup_scripts/` | Older environment setup helpers for Ubuntu, WSL, and VM workflows |

## Local-Only Archive

`untracked_materials/` is ignored by git. It is used for local-only files such
as old slide decks, rendered document images, patch files, temporary editor
state, and historical result snapshots.
